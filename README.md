# 新装机工具合集，持续更新（Ubuntu version>17.04）
## 用法

```shell
sh -c $(wget https://raw.githubusercontent.com/timeliar/my-ubuntu-installation/master/install.sh -O -)
```
## 结构

```shell
cmds/				---- 自定义可执行脚本
view/				---- 外观相关
	|--themes/		---- 主题
	|--icons/		---- 图标
	|--fonts/		---- 字体
install.sh			---- 一键安装脚本
```
